# The Boss Agency - Website Una página web moderna, audaz e innovadora para The Boss Agency, una agencia de marketing de clase mundial. Diseñada con tipografía impactante, animaciones suaves y un enfoque 100% en resultados medibles. ## 🎨 Características Principales - **Tipografía Audaz**: Títulos gigantes en Poppins Black para máximo impacto visual - **Diseño Innovador**: Inspirado en Sessions.co.uk con layout asimétrico y dinámico - **Carrusel Interactivo**: Casos de éxito con navegación fluida y números destacados - **Números Grandes**: Estadísticas y resultados prominentes en cada sección - **Paleta Premium**: Blanco, negro y dorado para transmitir lujo y profesionalismo - **Animaciones Suaves**: Transiciones fluidas y hover effects elegantes - **Responsive Design**: Optimizado para móvil, tablet y desktop - **Formulario de Contacto**: Integrado en sección oscura con validación ## 🛠️ Stack Tecnológico - **React 19**: Framework de UI - **TypeScript**: Tipado estático - **Tailwind CSS 4**: Estilos utilitarios - **Vite**: Bundler y dev server - **Framer Motion**: Animaciones (preparado para implementar) - **Lucide React**: Iconografía - **Wouter**: Enrutamiento cliente ## 📁 Estructura del Proyecto ``` thebossagency-website/ ├── client/ │ ├── public/ # Archivos estáticos │ │ └── __manus__/ # Configuración de Manus │ └── src/ │ ├── components/ # Componentes React │ │ ├── Header.tsx │ │ ├── Hero.tsx │ │ ├── Services.tsx │ │ ├── Portfolio.tsx │ │ ├── About.tsx │ │ ├── Contact.tsx │ │ ├── Footer.tsx │ │ └── ui/ # Componentes shadcn/ui │ ├── pages/ # Páginas │ ├── contexts/ # React Contexts │ ├── hooks/ # Custom Hooks │ ├── lib/ # Utilidades │ ├── App.tsx # Componente raíz │ ├── main.tsx # Entrada de React │ └── index.css # Estilos globales ├── server/ # Backend (placeholder) ├── shared/ # Código compartido ├── package.json ├── vite.config.ts ├── tsconfig.json └── README.md ``` ## 🚀 Instalación y Desarrollo ### Requisitos Previos - Node.js 18+ - pnpm (recomendado) o npm ### Instalación ```bash # Clonar el repositorio git clone https://github.com/tuusuario/thebossagency-website.git cd thebossagency-website # Instalar dependencias pnpm install # o npm install ``` ### Desarrollo ```bash # Iniciar servidor de desarrollo pnpm dev # o npm run dev ``` El servidor estará disponible en `http://localhost:5173` ### Build para Producción ```bash # Compilar para producción pnpm build # o npm run build # Vista previa del build pnpm preview # o npm run preview ``` ## 📋 Secciones de la Página ### 1. **Header** Navegación fija con logo, menú de navegación y botón CTA. Responsive con menú hamburguesa en móvil. ### 2. **Hero Section** Título gigante "HACEMOS QUE LAS MARCAS DOMINEN EL MERCADO" con imagen integrada. Estadísticas clave y dos CTAs principales. ### 3. **Services (Tres Pilares)** Tres servicios principales en sección dorada: - Estrategia Digital (250% engagement) - Marketing Orientado a Resultados (5M+ impresiones) - Gestión de Marca (100K+ seguidores) ### 4. **Portfolio (Casos de Éxito)** Carrusel interactivo de proyectos con: - Imagen del proyecto - Estadísticas de éxito (250%, 5M+, 180%, 100K+) - Navegación con botones y indicadores - Contador de slides ### 5. **About (Quiénes Somos)** Sección en fondo negro con: - Descripción de la agencia - Estadísticas destacadas (150+, 3.5X, 98%, 12+) - Valores de la empresa - Sección de equipo experto ### 6. **Contact (Contacto)** Formulario de contacto con: - Información de contacto (email, teléfono, ubicación) - Formulario con validación - Diseño en fondo blanco y negro ### 7. **Footer** Enlaces de navegación, servicios, redes sociales y información legal. ## 🎨 Paleta de Colores | Color | Hex | Uso | |-------|-----|-----| | Blanco | #FFFFFF | Fondo principal | | Negro | #000000 | Tipografía principal | | Dorado | #D4AF37 | Acentos, botones, destacados | | Gris Oscuro | #1a1a1a | Fondos alternados | ## 📱 Tipografía - **Poppins**: Títulos (Bold/Black) - **Montserrat**: Subtítulos y etiquetas (Bold/SemiBold) - **Inter**: Body text (Regular) ## 🔧 Configuración ### Variables de Entorno Crear archivo `.env.local` en la raíz del proyecto: ```env VITE_APP_TITLE=The Boss Agency VITE_APP_ID=thebossagency ``` ### Tailwind CSS La configuración de Tailwind está en `tailwind.config.js`. Los colores personalizados (dorado, etc.) están definidos en `client/src/index.css`. ## 📊 Performance - **Lighthouse Score**: 90+ - **Bundle Size**: ~150KB (gzipped) - **Time to Interactive**: <2s - **Responsive**: Mobile-first approach ## 🔐 Seguridad - Validación de formularios en cliente y servidor - Sanitización de inputs - CORS configurado correctamente - Dependencias actualizadas regularmente ## 📝 Componentes Principales ### Header.tsx Navegación con scroll suave entre secciones. Responsive con menú hamburguesa. ### Hero.tsx Sección hero con tipografía gigante, imagen integrada y CTAs. ### Services.tsx Tres pilares de servicios en sección dorada con iconografía. ### Portfolio.tsx Carrusel interactivo de casos de éxito con navegación. ### About.tsx Información sobre la agencia, valores y equipo. ### Contact.tsx Formulario de contacto con información de contacto. ### Footer.tsx Enlaces, redes sociales e información legal. ## 🚀 Deployment ### Opción 1: Manus (Recomendado) ```bash # Publicar directamente desde Manus # Usa el botón Publish en el Management UI ``` ### Opción 2: Vercel ```bash # Instalar Vercel CLI npm i -g vercel # Deploy vercel ``` ### Opción 3: Netlify ```bash # Instalar Netlify CLI npm i -g netlify-cli # Deploy netlify deploy --prod --dir=dist ``` ### Opción 4: GitHub Pages ```bash # Configurar en vite.config.ts # base: '/thebossagency-website/' pnpm build # Subir carpeta dist a gh-pages branch ``` ## 📚 Documentación Adicional - [React Documentation](https://react.dev) - [Tailwind CSS](https://tailwindcss.com) - [Vite Guide](https://vitejs.dev) - [TypeScript Handbook](https://www.typescriptlang.org/docs) ## 🤝 Contribuciones Las contribuciones son bienvenidas.
